# standards-version-tracker

Digital Identity 표준 버전 추적 저장소

## 개요
이 저장소는 Digital Identity 관련 주요 표준들의 Stable 및 Draft 버전을 추적합니다.

- **CSV 파일**: [standards.csv](./standards.csv)
- **작업 지침**: [GUIDELINES.md](./GUIDELINES.md)
