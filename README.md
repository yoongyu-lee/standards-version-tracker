# standards-version-tracker

Digital Identity 표준 버전 추적 저장소

## 개요
이 저장소는 Digital Identity 관련 주요 표준들의 Stable 및 Draft 버전을 추적합니다.

- **CSV 파일**: [standards.csv](./standards.csv)
- **작업 지침**: [GUIDELINES.md](./GUIDELINES.md)

## 변경 내역

### 2026-01-15
- 12개 표준 전체 최신 버전 정보 업데이트
- 주요 변경사항:
  - **W3C VC Data Model**: v2.0 Recommendation 공개 (2025-05-14), v2.1 Editor's Draft 진행 중
  - **OIDF OID4VCI**: v1.0 Final 승인 (2025-09-15)

- 모든 표준의 '핵심 변경 내용' 컴럼 작성 완료
